# Polymarket India — project rules

If anything here conflicts with `AGENTS.md`, follow `AGENTS.md`.

## Repo conventions

- Next.js App Router: `src/app/`
- API routes: `src/app/api/**/route.ts`
- Tests: `src/tests/` (`npm test`)

## Harness parity

- Use the same harness skills as Claude: `.agents/skills/*/SKILL.md`.
- When `CLAUDE.md` mentions a skill name, the matching implementation should exist under `.agents/skills/`.
